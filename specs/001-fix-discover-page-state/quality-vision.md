# 质量目标

## 用户可见行为

- 新建节目必须是干净状态，不显示其它节目的新闻、候选或已选素材。
- 发现页采集动作必须有结束态：成功、失败或超时，不允许按钮无限 loading。
- 依赖外部能力的开关必须先判断配置条件，不能假开启。
- 按钮图标语义必须和 tooltip 一致，“返回”使用返回箭头。

## 技术边界

- 节目内容归属 workflow state。
- 全局雷达缓存只作为采集服务缓存，不直接作为某个节目的发现页数据源。
- 本次不重构雷达服务、不改 fetch 节点协议。

## 回归风险

- 采集结果从全局缓存改写入当前 workflow 后，需确认发现页列表、进入整理、清空新闻都仍能同步页面状态。
- 返回图标替换需避免影响关闭节目、删除素材、关闭弹窗等真正关闭动作。
