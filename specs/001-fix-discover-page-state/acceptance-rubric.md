# 验收 Rubric

| 项目 | 通过标准 |
| --- | --- |
| 节目隔离 | 新建节目后发现页新闻列表为空，不继承旧节目或全局雷达缓存 |
| 采集按钮 | 点击“立即采集”后最终恢复可点击状态，并显示成功、失败或超时提示 |
| 智能标签 | 未配置 LLM 时点击智能标签不会开启，并给出配置引导 |
| 清空新闻 | 清空后发现页不刷控制台 `fetchContents:` 日志，当前节目相关发现状态被清空 |
| 返回图标 | 发现、整理、构思、写作、制作、发布页顶部返回按钮均为返回箭头 |
| 设置图标 | 雷达设置入口使用设置图标 |
| 构建 | `npm run verify` 和 `npm run build` 通过 |
